Disable Application Shortcuts

Some built-in applications can take up global shortcuts and cause conflicts, e.g. AMD Control Panel takes up Ctrl + Shift + L shortcuts, NVDIA graphics programs, etc. They should be disabled.

Remap Modifier Keys

Use MacOS-style ...

This schema is conducive to improving typing and triggering shortcuts, and reducing muscle strain.

  • New order like Win Alt Ctrl
  • Swap CapsLock and Esc

Enable: (RunAs Admin)

reg add "HKLM\SYSTEM\CurrentControlSet\Control\Keyboard Layout" /v "Scancode Map" /t REG_BINARY /d 00000000000000000600000001003A003A0001001D0038005BE01D0038005BE000000000 /f

Disable: (RunAs Admin)

reg delete "HKLM\SYSTEM\CurrentControlSet\Control\Keyboard Layout" /v "Scancode Map" /f

Log out and log in to take effect.

Sticky keys

Press shift key 5 times to activate.

Optimize: press modifier key, once to activate, twice to lock, three times to simulate down and up 

reg add "HKCU\Control Panel\Accessibility\StickyKeys" /v Flags /t REG_SZ /d 190 /f
